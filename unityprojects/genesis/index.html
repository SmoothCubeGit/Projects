Devlog: Genesis - 2D/3D Level Creation Tool

Overview

Genesis is a game creation tool designed to offer a unique platform for users to build, play, and share their own levels in a Mario Maker-like environment. The project focused primarily on the building system and UI functionality, providing a framework for level creation and gameplay. Although sharing functionality wasn't fully implemented, the tool allowed for an intuitive and enjoyable game creation experience.

Features and Systems Created

Building System

  • UI Elements: Utilized a UI pack from Kenney, custom fonts from DaFont, and a custom logo to design an attractive user interface.
  • Object Creation: Enabled both simple and complex object creation, allowing users to easily design their levels with various objects.
  • Platformer Movement: Developed a player movement system inspired by platformer games, allowing for smooth interactions with created levels.
  • Custom AI: Implemented a Goomba-like AI to populate levels with enemies.

Gameplay Features

  • Camera and Lighting: Integrated a camera system that adjusts based on gameplay and enhanced lighting effects using Unity's URP, which added bloom effects for a more atmospheric experience.
  • Spawnpoint and Death System: Created systems for player spawning and death management to ensure proper gameplay flow during level playthroughs.
  • Saving and Loading: Developed a framework to save and load objects between build and play modes, making it easier for users to switch between designing and testing levels.
  • Custom Properties: Added properties like "Has Rigidbody" and "Has AI" to objects, giving them additional functionality in the game world.

Audio and Visual Effects

  • Sound Effects: Implemented walking sound effects to enhance the player's movement experience.
  • Animations: Used animations from Mixamo to provide character and object movements within the game.
  • Triplanar Textures: Used triplanar texturing for dynamic surface mapping to objects within the level.

Development Highlights

Most Satisfying Achievements

  1. Lighting Effects: The use of Unity's URP to create bloom effects for lighting was particularly rewarding and helped to elevate the overall visual quality.
  2. Building System: The intuitive building system made it easy for users to design and create levels quickly, which was a key feature of the project.

Challenges and Improvements

  1. Atmospheric Sound Effects: The game could have benefited from a more extensive set of sound effects to create a richer atmosphere.
  2. Object Selection: More objects and assets could be added to the game creation system to provide greater variety and flexibility for users.

Lessons Learned

This project provided invaluable experience in:

  • Building a flexible 2D/3D level creation tool.
  • Implementing custom AI and player movement mechanics for a platformer-style game.
  • Creating a user-friendly UI system to facilitate level creation and testing.

Conclusion

Genesis successfully achieved its goal of creating a level creation tool with a working build and play system, although there is room for improvement, particularly in terms of adding more assets and sound effects. The project provided valuable learning opportunities in game development and UI design, offering a solid foundation for future enhancements and sharing capabilities.
