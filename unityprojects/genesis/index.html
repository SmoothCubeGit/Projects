Devlog: Genesis - 2D/3D Level Creation Tool

Overview

Genesis is a creative design tool designed to offer a unique platform for users to build, test, and share their own projects in a way similar to other creative construction tools. The project focused primarily on the construction system and interface functionality, providing a framework for creation and exploration. Although sharing functionality wasn't fully implemented, the tool allowed for an intuitive and enjoyable creation experience.

Features and Systems Created

Building System

  • UI Elements: Utilized a UI pack from Kenney, custom fonts from DaFont, and a custom logo to design an attractive user interface.
  • Object Creation: Enabled both simple and complex object creation, allowing users to easily design their environments with various objects.
  • Platformer Movement: Developed a movement system inspired by traditional physics-based mechanics, allowing for smooth interactions with created spaces.
  • Custom AI: Implemented an AI system to populate spaces with interactive elements.

Gameplay Features

  • Camera and Lighting: Integrated a camera system that adjusts based on activity and enhanced lighting effects using Unity's URP, which added bloom effects for a more atmospheric experience.
  • Spawnpoint and Reset System: Created systems for object placement and reset management to ensure proper flow during project interactions.
  • Saving and Loading: Developed a framework to save and load objects between build and test modes, making it easier for users to switch between designing and testing creations.
  • Custom Properties: Added properties like "Has Rigidbody" and "Has AI" to objects, giving them additional functionality in the design world.

Audio and Visual Effects

  • Sound Effects: Implemented walking sound effects to enhance user interaction.
  • Animations: Used animations from Mixamo to provide movement within the project space.
  • Triplanar Textures: Used triplanar texturing for dynamic surface mapping to objects within the environment.

Development Highlights

Most Satisfying Achievements

  1. Lighting Effects: The use of Unity's URP to create bloom effects for lighting was particularly rewarding and helped to elevate the overall visual quality.
  2. Building System: The intuitive building system made it easy for users to design and create quickly, which was a key feature of the project.

Challenges and Improvements

  1. Atmospheric Sound Effects: The creation could have benefited from a more extensive set of sound effects to create a richer atmosphere.
  2. Object Selection: More objects and assets could be added to the creation system to provide greater variety and flexibility for users.

Lessons Learned

This project provided invaluable experience in:

  • Building a flexible 2D/3D creation tool.
  • Implementing custom AI and movement mechanics for a simulation-style experience.
  • Creating a user-friendly UI system to facilitate creation and testing.

Conclusion

Genesis successfully achieved its goal of creating a creation tool with a working build and test system, although there is room for improvement, particularly in terms of adding more assets and sound effects. The project provided valuable learning opportunities in design and UI development, offering a solid foundation for future enhancements and sharing capabilities.
